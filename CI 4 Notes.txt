7. To create layout you need a main base file which will contain all design of a view. We pass a placeholder for content here with this code (Filename: layout/base.php): 

<?= $this->renderSection('content') ?>


8. To render this base.php design in actual view. We extend with this code:
<?= $this->extend('layout/base') ?>


9. After installing base theme. We need to render content inside the view. We wrap our code in this:
<?= $this->section('content') ?> 	--> wrapper need to be used
    <h1>Hello World!</h1>  		--> actual content
<?= $this->endSection() ?>


10. And you can do this process to many section of contents. This way you can divide the design and dynamic content part.


10. Small widgets, or view files can be included via the Partials:

12. To generate SEO friendly and readable url. We use routing in CI. Go to APP\CONFIG\routes.php. Here you set new routes with following syntax:

13. Using routes you can make long urls into short one with passing diffrent condition as well. (:num) makes sure we are passing only number. It's called Placeholder.
$routes->get('/test/(:num)', 'Welcome::test/$1');

17. Model are PHP Class file which contains methods to interact with DB itself. We create all our model files inside APP\MODELS.  Models always return the data to Controller who then pass this data into views.
==============================
Showmodel.php ------- Model
==============================
<?php
namespace App\Models;
use CodeIgniter\Model;

class Showmodel extends Model {

  public function getData() {
    $db = \Config\Database::connect();		--> Connect to default DB

    $builder = $db->table('users');		--> Connecting to User table with Builder
    $query   = $builder->get();			--> Getting result

    return $query->getResult();			--> returning the results back to controller
  }
}


18. Next we need to connect our model to controller and then access our data fetching method of model
==================================
Showdata.php ------ Controller
==================================
<?php
namespace App\Controllers;
use \App\Models\Showmodel;			--> Accessing Model in controller

class Showdata extends BaseController
{
  public function index()
  {
    $showmodel = new Showmodel();		--> Object of Model class to access its methods

    $data['users'] = $showmodel->getData();	--> Holding data in assoc array

    return view('showdata', $data);		--> Passing the data into view
  }
}
