1. To remove the public part from url and directly access it. we simply copy and paste .htaccess and index.php from public folder into main folder. And do set your path in index.php

2. We can use closure(anoynoumus function) to call views directly from routes.
$routes->get('/about', function(){
    echo view('about');
});

3.When creating controller always define controller with capitalize first letter. this is the standard way. Even though in smallcase it works

4. Creating model is same as controller.

5. To show hidden error. we need to save env file to .env file and uncomment the line and change production to development
# CI_ENVIRONMENT = development

6. For inserting data from form using the builder way
public function insert()
    {
    	$password = $this->request->getVar('password');

    	$data = ['password' => $password];

    	$db = \Config\Database::connect();
		$builder = $db->table('users');

		$builder->insert($data);
    }
